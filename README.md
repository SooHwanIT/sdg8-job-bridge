# JobBridge 💼

> 회사와 알바를 **완전히 분리한** 채용 플랫폼 — 근로기준법 규칙엔진이 계약서를 검토합니다

**SDG 목표:** SDG 8 — 양질의 일자리와 경제성장 (Decent Work and Economic Growth)

## 핵심 차별점

**계약서 AI 검토 = 진짜 근로기준법 규칙엔진**
손으로 적은 결과가 아니라, 정형 계약 필드에 **12개 근로기준법 규칙**을 실행해 판정을 만들고, **근거 조문 인용 + 준수 점수(100점) + 등급**을 산출합니다. 외부 API·키 없이 클라이언트에서 동작합니다.

- **최저임금**(2025 · 10,030원) · **주휴수당**(주 15h) · **연장 150%/야간·휴일 50% 가산** · **수습 감액 하한 90%·1년 이상** · **위약예정 금지(§20)** · **재량근로 대상 제한(§58③)** · **서면 근로계약서 교부(§17)** 등
- **내 계약서 직접 검토**: 시급·근무시간·수당을 입력하면 **실시간** 판정 (`ContractChecker`)

## 주요 기능

- **회사 / 알바 완전 분리**: 모드를 고르면 해당 공고만 노출. 각 고용형태에 맞는 권리만 안내.
- **공고 · 상세 · 리뷰**: 검색, 연봉/시급·가산수당·복지, 별점 리뷰(작성 가능).
- **AI 계약서 검토**: 위 규칙엔진. 공고 카드에도 계약 점수 배지 표시.
- **기업 광고 · 쇼츠**: 모드별 영상만 노출(좋아요·저장·구독).
- **메시지 & 전화**: 빠른 질문 템플릿, 타이핑 인디케이터.
- **마이페이지**: 나이스(NEIS) 성적·출결·자격증 불러오기, **AI 자소서 초안**(어조·지원대상 반영, 편집 가능), 원클릭 지원 현황.
- **임팩트 탭**: SDG 8 세부 타깃 매핑(8.3/8.5/8.6/8.8).

## 기술 스택

- Vite 6 + React 18 · Tailwind CSS v4 (`@tailwindcss/vite`)
- 규칙엔진: `src/lib/laborLaw.js`(룰셋) + `src/lib/contractEngine.js`(평가)
- 상태 저장: 버전드 `localStorage` (시크릿모드 안전, 마이그레이션 지원)
- 반응형: 데스크톱 사이드바 / 모바일 하단탭, 접근성(aria·키보드·reduced-motion)
